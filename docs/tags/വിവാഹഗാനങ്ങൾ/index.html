ബാബുവിന്റെ ഗാനശേഖരം.

വിവാഹഗാനങ്ങൾ

  • ആദിയിൽ അഖിലേശൻ
  • ഈ ഭൂമിയിൽ സ്വർഗ്ഗം തീർത്തിടുവാനായ്
  • കുടുംബം ദൈവത്തിൻ കുടുംബം
  • കൂടുന്നോരിമ്പം കുടുംബം
  • കൂടുമ്പോൾ ഇമ്പമുള്ള കുടുംബം
  • കൂടുമ്പോൾ ഇമ്പമേറും കുടുംബം
  • പുതിയൊരു പുലരിയിതാ
  • മംഗല്യ സൗഭാഗ്യമേകാൻ

    • നിർമ്മിച്ചത്:
    തിയോഫിൻ സാവിയോ തിയഡോർ

    പൂർണ്ണമായും സ്വതന്ത്ര സോഫ്റ്റ്‍വെയർ
