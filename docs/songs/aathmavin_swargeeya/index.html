(ആത്മാവിൻ…)

ക്ഷീണവും ദാഹവും തീർത്തു നീ….(2)
ആത്മവീര്യം വളർത്തുന്നു
(ആത്മാവിൻ…)

ജീവനും ശാന്തിയും നൽകി നീ…..(2)

(ആത്മാവിൻ…)
