പാപമോചന സമയം
സമർപ്പണത്തിൻ സമയം (2)

രാജ രാജ രാജനേ ക്രിസ്തുരാജനേ.. (2)

എന്നന്തരാത്മാവിൽ നീ നിറയുമ്പോൾ
മനസ്സൊരു തൂമഞ്ഞുതുള്ളി പോലെ (2)

തിരുവചനങ്ങൾ പാടുമ്പോൾ
അനുഗ്രഹപ്പൂമഴ ചൊരിയണമേ (2)

രാജ രാജ രാജനേ ക്രിസ്തുരാജനേ..
രാജ രാജ രാജനേ ക്രിസ്തുരാജനേ.. (2)

ഒരു തിരിനാളമായി തീർന്നിടും ഞാൻ (2)
