നാഥാ ഈ ബലിവേദിയിൽ
കാണിയ്കയായ് എന്നെ നല്കുന്നു ഞാൻ (2)

അന്നാ കാൽവരി മലമുകളിൽ
തിരുനാഥൻ ഏകിയ ജീവാർപ്പണം (2)
പുനരർപ്പിക്കുമീ തിരുവൾത്താരയിൽ
അണയാം ജീവിത കാഴ്ച്ചയുമായ്‌ തിരുമുൻപിൽ (2)
